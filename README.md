* dependecies
```
node = v18.9.0
npm = 8.19.1
```

* how to get server up and running
```
cd server
npm install
node index.js
go to:  http://localhost:8080/  and should see server running
```

* how to get front end up and running
```
cd frontend
npm install
npm start

go to: http://localhost:3000/ 
```
